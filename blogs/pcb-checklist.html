PCB Design checklist before manufacturing

A simple checklist to avoid common PCB mistakes.

KiCadPCBManufacturing

Checklist

1) Run DRC (no errors / clearance ok)
2) Verify footprints (pin order, polarity, dimensions)
3) Check power rails (5V/3V3, current paths, thickness)
4) Place decoupling capacitors close to IC pins
5) Grounding: clean return paths, avoid long loops
6) Silkscreen: labels, polarity marks, pin-1 markers
7) Add test points for key rails and signals
8) Check connectors orientation and mechanical spacing
9) Export gerbers and verify in a viewer
10) BOM + assembly notes (if PCBA)

My rule

Always verify footprints using the datasheet + 3D view. Most PCB issues come from a wrong footprint or swapped pins.
