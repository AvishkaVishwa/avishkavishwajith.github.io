Sumo Robot (Competition)

Autonomous sumo robot built on ESP32 with TB6612FNG motor driver and competition-ready control logic.

ESP32RoboticsMotor ControlCompetition

Overview

A compact autonomous robot designed for sumo competitions, focused on quick response and stable traction.

Goals

  • Fast detection of opponent and ring edge
  • Strong push control with smooth motor drive
  • Reliable behavior under competition stress

Hardware

  • ESP32 controller
  • TB6612FNG motor driver
  • Sensor array for detection and edge awareness

Control Logic

  • Sensor polling and decision state machine
  • Motor speed tuning for traction
  • Failsafe behavior near ring edge

Challenges and Fixes

The biggest challenge was tuning motor response against different surfaces. Iterative tests improved stability.

Results

The robot achieved 1st runners-up at UoC FOT 2024, validating the control strategy and hardware design.
