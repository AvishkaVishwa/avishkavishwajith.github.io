Step Counter with Fall Detection

Wearable step counter and fall detection system using ESP32, MPU6050, and an OLED status display.

ESP32MPU6050OLEDESP-IDF

Overview

A compact wearable system that tracks steps and detects falls using IMU data, with live status shown on an OLED.

Goals

  • Reliable step counting with simple signal processing
  • Basic fall detection using thresholds and timing
  • Clear on-device feedback via OLED

Hardware

  • ESP32 microcontroller
  • MPU6050 accelerometer and gyro
  • OLED display and battery power

Firmware

  • IMU sampling and filtering
  • Step detection using acceleration magnitude
  • Fall detection based on impact thresholds and recovery time

Challenges and Fixes

Motion noise required filtering and careful threshold tuning. Iterative testing improved the balance between sensitivity and false positives.

Results

The device detects daily walking steps and flags heavy impacts for fall review. The logic can be refined further with more data.
