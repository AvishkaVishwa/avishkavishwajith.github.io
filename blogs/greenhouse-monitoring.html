Greenhouse Monitoring System

ESP8266 based greenhouse monitoring with soil moisture, MQ-135, DHT11, and LDR sensors streamed to a Blynk dashboard for real-time visibility.

ESP8266BlynkIoTSensors

Overview

This project focuses on low-cost monitoring for greenhouse conditions. The system collects environmental data and publishes it to a mobile dashboard for quick checks and alerts.

Goals

  • Track temperature, humidity, air quality, soil moisture, and light level.
  • Send data to a remote dashboard with simple alerts.
  • Keep the electronics simple and easy to replicate.

Hardware

  • ESP8266 WiFi module
  • DHT11, MQ-135, soil moisture probe, LDR
  • Regulated 5V to 3.3V power stage

Firmware and Cloud

  • Periodic sensor sampling with basic filtering
  • Mapped sensor data to Blynk virtual pins
  • Connection retry logic to keep WiFi stable

Challenges and Fixes

Sensor noise and WiFi drops were the main issues. Averaging readings and adding reconnect logic improved stability for long runtime monitoring.

Results

The dashboard shows live readings with clear trends. The setup is ready to expand with actuators such as fans or pumps.
