Temperature Monitoring System

ATmega328P temperature monitor with interrupt-driven timing and a 4-digit seven-segment display for real-time readings.

ATmega328PAVR C7-segmentEmbedded

Overview

This project demonstrates a clean embedded design using interrupts to keep timing accurate while driving a multiplexed display.

Goals

  • Stable, flicker-free 7-segment display output
  • Consistent temperature sampling and conversion
  • Simple user interface for continuous monitoring

Hardware

  • ATmega328P microcontroller
  • Analog temperature sensor
  • 4-digit seven-segment display

Firmware

  • Timer interrupts for display multiplexing
  • ADC sampling with simple averaging
  • Numeric formatting for 7-seg output

Challenges and Fixes

Display flicker and noisy readings were reduced by tuning refresh rates and adding ADC averaging.

Results

The monitor provides a clear real-time temperature readout and a solid base for future sensor expansion.
