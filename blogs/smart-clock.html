ESP32 Powered Smart Clock

ESP32 smart clock with SNTP sync, MAX7219 display, web configuration, and a custom power PCB.

ESP32MAX7219PCBESP-IDF

Overview

A connected clock that syncs time via SNTP and offers web-based configuration for alarms and timers.

Goals

  • Reliable time sync over WiFi
  • Clean display output with MAX7219
  • Custom PCB for compact integration

Hardware

  • ESP32 module with WiFi
  • MAX7219 4-digit display
  • Custom power chain using LM2596S and AMS1117

Firmware

  • SNTP time sync and local timekeeping
  • Web UI for configuration
  • Alarm and timer scheduling

Challenges and Fixes

Power stability and display refresh timing required tuning. Better decoupling and update scheduling solved flicker.

Results

The clock keeps accurate time and exposes settings over a simple web interface.

  • Framework: ESP-IDF
